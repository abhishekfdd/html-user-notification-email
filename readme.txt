This plugin gives you option for sending html new user notification mail.

== Description ==

This plugin allows to send html new user notification emails in WordPress.

This plugin was created by Abhishek Kumar http://github.com/abhishekfdd

== Installation ==

1. Install Disable Html New User Notification either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate Disable Html New User Notification.
